Anlisis Exploratorio del Dataset de Iris

Objetivo del Proyecto

El objetivo de este anlisis es explorar el famoso dataset de sklearn para entender las caractersticas de las diferentes especies de flores. Buscaremos identificar patrones en las medidas de los ptalos y spalos que nos permitan distinguir entre las tres especies: setosa, versicolor y virginica. Realizaremos un resumen estadstico y visualizaciones clave para lograr este entendimiento.

Visualizaciones

A continuacin, presentamos dos visualizaciones para explorar las relaciones entre las variables y las especies.

  1. Grfico de dispersin (Scatter Plot): Muestra la relacin entre la longitud y el ancho de los spalos, coloreados por especie. Esto nos ayudar a ver si hay agrupaciones visuales.
  2. Grfico de violn (Violin Plot): Muestra la distribucin de la longitud de los ptalos para cada especie, ofreciendo una visin de la densidad y el rango de valores.

Limpieza y Transformacin de Datos

El dataset de Iris ya est notablemente limpio y no requiere una limpieza exhaustiva. Sin embargo, hemos realizado una transformacin clave para facilitar el anlisis:

  • Transformacin de la columna target: La columna numrica original target ha sido transformada a la columna species, que contiene los nombres de las especies (setosa, versicolor, virginica). Esto hace que el anlisis sea ms intuitivo y las visualizaciones ms informativas. No se detectaron valores nulos o atpicos significativos que requieran una limpieza adicional.

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris

# Cargar el dataset de Iris
iris = load_iris()
df = pd.DataFrame(data=iris.data, columns=iris.feature_names)
df['species'] = iris.target_names[iris.target]

# Resumen estadstico
print("Resumen estadstico del dataset:")
print(df.describe())

# Ver las primeras 5 filas
print("\nPrimeras 5 filas del dataset:")
print(df.head())

# Visualizacin 1: Scatter plot
plt.figure(figsize=(10, 6))
sns.scatterplot(x='sepal length (cm)', y='sepal width (cm)', hue='species', data=df)
plt.title('Longitud vs. Ancho del Spalo por Especie')
plt.xlabel('Longitud del Spalo (cm)')
plt.ylabel('Ancho del Spalo (cm)')
plt.show()

# Visualizacin 2: Violin plot
plt.figure(figsize=(10, 6))
sns.violinplot(x='species', y='petal length (cm)', data=df)
plt.title('Distribucin de la Longitud del Ptalo por Especie')
plt.xlabel('Especie')
plt.ylabel('Longitud del Ptalo (cm)')
plt.show()
