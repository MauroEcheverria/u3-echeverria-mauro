Construccin de un Modelo de Clasificacin Base

Objetivo del Modelo

El objetivo de esta seccin es utilizar los datos limpios para entrenar un modelo de machine learning que pueda predecir la especie de una flor basndose en sus medidas de spalo y ptalo. Esto nos permitir validar si las diferencias observadas en el anlisis exploratorio son lo suficientemente significativas para una clasificacin automtica.

Explicacin del Modelo y Mtricas

Se ha elegido un modelo de clasificacin K-Nearest Neighbors (KNN), que es un algoritmo simple pero eficaz. Su funcionamiento se basa en la proximidad: predice la especie de una nueva flor en funcin de las especies de sus 'k' vecinos ms cercanos.

Para evaluar el rendimiento del modelo, utilizamos la precisin (accuracy) como mtrica. La precisin nos indica el porcentaje de predicciones correctas que el modelo realiz en un conjunto de datos que no vio durante el entrenamiento (conjunto de prueba). Una alta precisin sugiere que el modelo ha aprendido con xito a diferenciar entre las tres especies de flores de Iris.

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris

# Cargar el dataset de Iris
iris = load_iris()
df = pd.DataFrame(data=iris.data, columns=iris.feature_names)
df['species'] = iris.target_names[iris.target]

# Histograma
plt.figure(figsize=(12, 6))
sns.histplot(data=df, x='sepal length (cm)', hue='species', kde=True, palette='viridis')
plt.title('Distribucin de la Longitud del Spalo por Especie')
plt.xlabel('Longitud del Spalo (cm)')
plt.ylabel('Frecuencia')
plt.show()

# Pair Plot
sns.pairplot(df, hue='species', palette='viridis')
plt.suptitle('Relaciones entre Variables por Especie', y=1.02)
plt.show()
